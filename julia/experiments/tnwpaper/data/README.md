# Experimental data — provenance and attribution

The experimental datasets in this folder are **digitisations of published figures**,
produced by the author (P. Ranganathan) for model–data comparison. They are the
extracted data points only; **no publisher PDFs, figures, or supplementary files are
redistributed here**. Each dataset is provided for transparency and reproducibility of
the analysis in the accompanying paper and should be cited to its original source below.

If you use any of these datasets, please cite the original publication, not just this
repository.

| Dataset (tag) | Digitised from | Citation |
|---|---|---|

## Notes

- **Method.** Data points were extracted from published figures by graphical
  digitisation. Small extraction error (point placement, axis calibration) is
  inherent; treat these as faithful reproductions, not the authors' original numerical
  data.
- **Licensing.** The digitised numerical values are released under the repository's MIT
  licence as the author's own extraction work. This does **not** relicense the original
  publications or their figures, which remain © their respective publishers. Attribution
  to the original source (above) is required.
- **Concentration conventions** differ by dataset (number-density `c/c*`, Martin/KM
  basis, Graessley good-solvent basis). See `PAPER_DATA.md` for the exact convention and
  per-sample parameters used for each dataset.
